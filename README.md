## 🎯 简介

ClawX 是基于 OpenClaw 的桌面 AI 助手应用，提供本地化的 AI Agent 体验。

### 核心特性
- ✅ 本地部署
- ✅ 隐私保护
- ✅ 多模型支持
- ✅ 技能扩展

---

## 📦 安装

### macOS
```bash
brew install clawx
```

### Windows
```bash
winget install clawx
```

### Linux
```bash
snap install clawx
```

---

## 🚀 快速开始

### 1. 启动 ClawX
```bash
clawx start
```

### 2. 配置模型
```bash
clawx config model
```

### 3. 安装技能
```bash
clawx skill install <skill-name>
```
